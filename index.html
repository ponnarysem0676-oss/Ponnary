Encryption & Decryption — Interactive Demo

Learn the basics of symmetric and asymmetric encryption and try them in your browser using the Web Crypto API.

Lesson: Basics

Encryption transforms readable data (plaintext) into an unreadable format (ciphertext). Decryption reverses it. Two common categories:

  • Symmetric encryption — same secret key used for encryption and decryption (example: AES-GCM).
  • Asymmetric encryption — public/private key pair. Public key encrypts, private key decrypts (example: RSA-OAEP).

This page uses the browser's window.crypto.subtle (Web Crypto API) for real cryptographic primitives. Do not use these demos directly for production secret management — they are for learning and prototyping only.

Generate an AES-256-GCM key, encrypt a short message, and then decrypt it.

Asymmetric demo — RSA-OAEP (browser)

Generate an RSA key pair, export the public key, encrypt with the public key, and decrypt with the private key.

Note: RSA can only encrypt small messages up to key-dependent limits. For large data use hybrid encryption (encrypt data with AES, encrypt AES key with RSA).

About this demo

This demo runs entirely in your browser using the Web Crypto API. Keys are generated locally and never leave your machine unless you copy and share them.
